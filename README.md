## Installation
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
cd backend\python
env\Scripts\activate
```

Got Auth
```bash
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

```bash
cd flip-disc_frontend
npm install
```
